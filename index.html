What I’m good at?

HTML & CSS

HTML (HyperText Markup Language) is a standardized system for marking up text documents to describe the structure and content of a webpage. It consists of a series of tags, or elements, that define the different parts of a document, such as headings, paragraphs, and links. HTML is used to create the basic structure and content of a webpage, and is usually paired with CSS (Cascading Style Sheets) to control the appearance and layout of the page.

MySql & PHP

PHP (Hypertext Preprocessor) is a server-side scripting language commonly used to develop dynamic websites and web applications. It allows developers to execute code on the server, generate dynamic HTML pages, and access and manipulate databases. PHP is often used in conjunction with MySQL. MySQL is a free, open-source database management system that is widely used to store and retrieve data for websites and applications.

JavaScript & Bootstrap

JavaScript is a programming language that is commonly used to build interactive, dynamic websites and web applications. It is a client-side language, which means that it is executed by the user's web browser rather than the server. JavaScript allows developers to create interactive features such as animations, form validation, and real-time updates, and is often used in combination with HTML and CSS to create rich, immersive user experiences.

C & Python

C is a general-purpose programming language that is widely used to develop a variety of software applications.
